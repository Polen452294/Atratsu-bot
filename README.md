# CargoLink

CargoLink — MVP-система для автоматизации обработки логистических лотов, подбора перевозчиков и подготовки данных для дальнейшей интеграции с внешними системами.

Проект реализован как backend-first решение на Python 3.12 с REST API и Telegram-ботом для логиста.

## Основная идея

Система принимает параметры выигранного лота, подбирает подходящих перевозчиков по заданным критериям, позволяет логисту выбрать лучший вариант и затем формирует экспорт в JSON или CSV для дальнейшей передачи в 1С или другую внешнюю систему.

На текущем этапе поиск перевозчиков реализован через mock-провайдер, чтобы можно было полноценно собрать и протестировать MVP без зависимости от внешнего API.

## Стек

- Python 3.12
- FastAPI
- PostgreSQL
- SQLAlchemy 2.x
- Alembic
- aiogram 3.x
- asyncpg
- Docker / Docker Compose

## Возможности MVP

### Backend API
- создание лота;
- просмотр списка лотов;
- получение лота по id;
- изменение статуса лота;
- запуск поиска перевозчиков;
- просмотр найденных кандидатов;
- выбор кандидата;
- создание сделки;
- инициация сделки;
- экспорт в JSON;
- экспорт в CSV;
- просмотр списка экспортов.

### Telegram-бот
- `/start`
- `/lot` — пошаговое создание лота через FSM;
- автоматический mock-поиск перевозчиков;
- выбор перевозчика через inline-кнопки;
- инициация сделки;
- `/my_lots` — просмотр последних лотов;
- экспорт JSON/CSV прямо из бота.

## Архитектура

Проект разделён на несколько слоёв:

- `api` — REST API на FastAPI;
- `bot` — Telegram-бот на aiogram;
- `services` — бизнес-логика;
- `repositories` — работа с БД;
- `integrations` — адаптеры внешних систем;
- `db` — SQLAlchemy модели и сессии;
- `domain` — Pydantic-схемы;
- `alembic` — миграции БД.
